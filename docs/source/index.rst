``flatstar`` is an open-source Python tool for drawing stellar disks as
``numpy.ndarray`` objects with scientifically-rigorous limb darkening. Each
pixel has an accurate fractional intensity in relation to the total stellar
intensity of ``1.0``. It is ideal for ray-tracing simulations of stars and
planetary transits.

using GitHub.
